Structure de base du HTML

La Structure du HTML est composé de 6 parties

  • Le doctype
  • La balise html
  • La balise head
  • La balise body
  • La balise footer

La balise body est composé de 3 parties :

  • La balise header
  • La balise main
  • La balise footer
